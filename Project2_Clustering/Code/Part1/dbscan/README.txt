-----------------------------2. INSTRUCTIONS---------------------
1. We use Python 3.6 and Spyder IDE to program.
2. To run DBSCAN algorithm, open 'dbscan.py' and directly run it. 
3. The default dataset is 'cho.txt', to alternate dataset, you can set manually.
4. The parameters used in DBSCAN algorithm are pre-determined using 'parameters_dbscan.py'.
5. The pre-determined parameters for the three datasets, 'cho', 'iyer', 'new_dataset_1', are already recorded in 'dbscan.py';
When alternating dataset, you can change the parameters according to the instruction in the comment in 'dbscan.py';
There's no need to open the 'parameters_dbscan.py' file if you just want to run the DBSCAN algorithm on the three datasets.
